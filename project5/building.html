Building the Homelab

Discover how I designed and built my homelab. See the hardware and software choices I made, and what I learned in the process.

Hardware

  • CPU/RAM sized for several VMs
  • SSD for OS + HDD/SSD for data
  • Reliable PSU and adequate cooling
  • Optional: small NAS or used mini-PCs

Networking

  • Network segmentation for different device types
  • Firewall rules to contain lab experiments
  • Separate guest/test networks
  • Basic performance/monitoring checks

Software Setup

  • Linux install + hypervisor (e.g., Proxmox)
  • Containers (e.g., Docker) for app deployment
  • Centralized updates/backups
  • Open-source software for experimentation
